根据**课件 08 (Connectionist)** 及相关课程笔记（Week 2-3），以下是关于神经网络基础理论的详细讲解：

### 1. 硬限幅、Sigmoid 与 MSE 损失

*   **硬限幅（Hard Limiting）**
    这是感知机最早使用的激活函数，也称为**阶跃函数（Step Function）** [1, 2]。它的输出是离散的（如 0/1 或 -1/1），其数学本质是一个**线性双极性阈值** [1, 3]。
    *   **局限性**：由于它在阈值点不连续且在其他地方导数为 0，导致无法利用微积分手段（如梯度下降）进行优化 [4]。

*   **Sigmoid（对数几率函数）**
    为了解决硬限幅不可微的问题，引入了**Sigmoid 激活函数** [4]。它是一个 S 形的连续曲线，能将输入映射到 $(0, 1)$ 区间 [1, 5]。
    *   **关键公式**：
        $$f(\sigma) = \frac{1}{1 + e^{-\sigma}}$$ [1, 5]
    *   **特性**：它在 0 附近近似线性，而在两端表现出**软阈值**特性，允许更精细的误差衡量 [6, 7]。

*   **MSE 损失（均方误差）**
    在训练网络时，用于衡量预测输出与期望值之间的差距 [8]。
    *   **关键公式**：
        $$Error = \frac{1}{2} \sum_{i} (d_i - O_i)^2$$ [1, 8]
        其中 $d_i$ 为目标值，$O_i$ 为网络实际输出。平方项确保了误差始终为正，且较大的误差会受到更重的惩罚 [8, 9]。

### 2. 梯度下降与误差曲面直觉

*   **误差曲面（Error Surface）**
    神经网络的误差曲面是由所有可能权重配置构成的 **n 维空间** [10]。
    *   **直觉形象**：线性模型的误差曲面是平滑的碗状，但由于神经网络引入了非线性激活函数，其曲面变得**极其崎岖**，类似于“被揉皱的纸” [11]。
    *   **局部极小值与鞍点**：在这样的曲面上，优化过程容易陷入**局部极小值（Local Minimum）**或停滞在**鞍点（Saddle Point）** [1, 11]。

*   **梯度下降（Gradient Descent）**
    这是一种寻找曲面“最低点”的迭代算法 [12, 13]。
    *   **核心逻辑**：在当前点计算梯度的反方向（下降最快方向），并沿着该方向移动一个微小的步长（由**学习率 $\eta$** 决定） [10, 11]。
    *   **更新规则**：
        $$\theta_{t+1} = \theta_t - \eta \nabla L(\theta_t)$$ [11]
    *   **下山比喻**：梯度下降法形象地被称为“下山法”，即在当前位置通过一阶估计寻找最快下降方向，并循环迭代直至收敛 [14]。

### 3. Sigmoid 导数曲线意义与饱和区

*   **导数公式**：
    Sigmoid 函数的一个优美特性是其导数可以用自身表示：
    $$f'(\sigma) = f(\sigma)(1 - f(\sigma))$$ [1, 15]

*   **饱和区（Saturation Zones）的物理意义**：
    Sigmoid 导数曲线呈钟形，在 $\sigma=0$ 时达到最大值（0.25），随着 $|\sigma|$ 增大，导数迅速趋于 0 [16, 17]。
    *   **饱和区定义**：当输入非常大或非常小时，函数进入“饱和区”，曲线变得非常平坦 [18, 19]。
    *   **梯度消失问题**：
        1.  由于导数因子 $f'(\sigma)$ 存在于权重更新公式中，如果神经元处于饱和区，**导数 $\approx 0$** [19, 20]。
        2.  这会导致权重更新停滞（即**梯度消失**），模型无法继续学习 [16, 21]。
    *   **训练建议**：为了避免饱和，通常建议将权重初始化为较小的随机值，使初始激活值落在 Sigmoid 的线性敏感区（导数较大的区域） [22, 23]。
