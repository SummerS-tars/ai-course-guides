在隐马尔可夫模型（HMM）中，**解码问题（Decoding Problem）**的目标是：给定观测序列 $O = \{O_1, O_2, \dots, O_T\}$ 和模型参数 $\mu = \{A, B, \Pi\}$，寻找一个最可能的隐状态序列 $X = \{X_1, X_2, \dots, X_T\}$，使得条件概率 $P(X|O, \mu)$ 最大化 [1-3]。

**维特比算法（Viterbi Algorithm）**是求解该问题的经典动态规划算法 [2, 4]。以下是该算法的核心逻辑、变量含义及步骤：

### 1. 核心变量定义

维特比算法维护两个核心变量，它们本质上是动态规划中的“状态”记录 [2, 5]：

*   **概率变量 $\delta_t(i)$（各路径最大概率）**：
    *   **含义**：在时刻 $t$，观测到序列 $O_1, \dots, O_{t-1}$（或 $O_t$ 视具体定义而定）且系统处于隐状态 $i$ 的所有可能路径中，概率最大的那条路径的概率值 [2, 3]。
    *   **物理意义**：它代表了“到达当前状态的最优局部得分” [5]。
*   **回溯指针 $\psi_t(i)$（路径记忆）**：
    *   **含义**：记录在时刻 $t$ 达到隐状态 $i$ 且取得最大概率 $\delta_t(i)$ 时，其在 $t-1$ 时刻的前驱状态编号 [2, 5]。
    *   **物理意义**：它像“面包屑”一样记录了最优路径的来源，用于最后的序列重构 [5]。

---

### 2. 算法步骤（递推与回溯）

根据课程课件和记录，维特比算法的执行流程如下 [2, 5, 6]：

1.  **初始化 (Initialization)**：
    *   对所有状态 $1 \le j \le N$，设定起始概率：
        $\delta_j(1) = \pi_j$ (注：部分教材会在此处乘以发射概率 $b_j(O_1)$)。
    *   初始时刻没有前驱状态，回溯指针通常设为空。

2.  **递推 (Induction/Recursion)**：
    *   对于时刻 $t = 1$ 到 $T$，以及每个当前状态 $j$，计算：
        $$\delta_j(t+1) = \max_{1 \le i \le N} \left[ \delta_i(t) \cdot a_{ij} \right] \cdot b_{j}(O_t)$$
    *   同时记录取得最大值的那个前驱状态 $i$：
        $$\psi_j(t+1) = \text{argmax}_{1 \le i \le N} \left[ \delta_i(t) \cdot a_{ij} \right]$$
    *   这一步实现了从“局部最优”推导“全局最优”的动态规划思想 [5]。

3.  **终止 (Termination)**：
    *   在最终时刻 $T$（或 $T+1$），在所有可能的结束状态中找到概率最大的那一个：
        $$\hat{X}_T = \text{argmax}_{1 \le i \le N} \delta_i(T)$$
    *   此时的最大概率即为整个最优路径的概率。

4.  **回溯 (Backtracking)**：
    *   根据记录的指针 $\psi$，从时刻 $T$ 开始逐层反推：
        $$\hat{X}_t = \psi_{\hat{X}_{t+1}}(t+1)$$
    *   最终得到完整的最优隐状态序列 $\hat{X} = \{\hat{X}_1, \hat{X}_2, \dots, \hat{X}_T\}$ [5, 6]。

---

### 3. 与前向算法的对比：「求和 vs 取 Max」

维特比算法与用于计算观测序列总概率的**前向算法（Forward Algorithm）**在结构上非常相似，均使用格点图（Trellis）进行计算，但数学操作有本质区别 [2, 5, 7]：

| 特性 | 前向算法 (Forward) | 维特比算法 (Viterbi) |
| :--- | :--- | :--- |
| **解决问题** | **评估问题**：计算 $P(O\|\mu)$ [1, 3] | **解码问题**：寻优 $I^* = \arg\max P(I\|O)$ [1, 3] |
| **核心操作** | **求和 ($\sum$)**：累加所有到达当前状态的路径概率 [3] | **取最大值 ($\max$)**：仅保留到达当前状态的最优路径概率 [5] |
| **物理直观** | **边缘化**：考虑所有可能的隐状态演化路径 [3] | **寻优**：从所有路径中挑出“冠军”路径 [5] |
| **输出结果** | 观测序列在模型下出现的总概率值 [7] | 一个确定的、概率最大的隐状态序列 [6] |

**对比总结**：前向算法通过**求和**实现了对隐状态的边缘化（Marginalization），从而获得整体似然；而维特比算法通过**取 Max** 实现了路径的贪心优选（实际上是全局最优的动态规划实现），从而获得最可能的解释 [2, 3, 5, 7]。
