### 确定性因子 (CF) 合成计算示例

**依据**：Week 13 课程记录（明确指出此为必考点）[1, 2]、课件 05 (Uncertainty) [3-6] 以及参考书 Luger [7, 8]。

---

### 1. 核心计算公式
在 MYCIN 风格的专家系统中，处理不确定性主要遵循以下规则：

*   **证据强度的逻辑组合** [4]：
    *   **AND (与)**：$CF(E_1 \cap E_2, e) = \min[CF(E_1, e), CF(E_2, e)]$
    *   **OR (或)**：$CF(E_1 \cup E_2, e) = \max[CF(E_1, e), CF(E_2, e)]$
    *   **NOT (非)**：$CF(\neg E, e) = -CF(E, e)$
*   **规则结论强度计算** [5, 9]：
    *   $CF(H, e) = CF(E, e) \times CF(H, E)$
    *   （其中 $CF(E, e)$ 为前提证据的观测强度，$CF(H, E)$ 为规则本身的置信度）
*   **合成公式 (Combining Function)** [6, 8]：
    *   当 $CF_1, CF_2 > 0$（**同正**）：$CF_{COMB} = CF_1 + CF_2(1 - CF_1)$
    *   当 $CF_1, CF_2 < 0$（**同负**）：$CF_{COMB} = CF_1 + CF_2(1 + CF_1)$
    *   当 **异号**（一正一负）：$CF_{COMB} = \frac{CF_1 + CF_2}{1 - \min(|CF_1|, |CF_2|)}$

### 2. 数值手算例子

**场景设定：**
现有三条规则指向同一个假设 $H$（例如“该生物为链球菌” [5]）：
1.  **规则 $r_1$**：IF $E_1$ AND $E_2$ THEN $H$ ($CF=0.8$)
2.  **规则 $r_2$**：IF $E_3$ THEN $H$ ($CF=0.5$)
3.  **规则 $r_3$**：IF $E_4$ THEN $H$ ($CF=-0.4$)

**当前观测到的证据强度：**
*   $CF(E_1, e) = 0.7$
*   $CF(E_2, e) = 0.6$
*   $CF(E_3, e) = 0.4$
*   $CF(E_4, e) = 0.5$

#### 步骤一：计算单条规则各自得出的结论强度

1.  **计算规则 $r_1$ 的结论强度**：
    *   前提证据强度（AND 逻辑）：$CF(E_{1 \cap 2}, e) = \min(0.7, 0.6) = 0.6$ [4]。
    *   应用规则置信度：$CF_1(H, e) = 0.6 \times 0.8 = \mathbf{0.48}$ [5]。
2.  **计算规则 $r_2$ 的结论强度**：
    *   结论强度：$CF_2(H, e) = 0.4 \times 0.5 = \mathbf{0.2}$ [5]。
3.  **计算规则 $r_3$ 的结论强度**：
    *   结论强度：$CF_3(H, e) = 0.5 \times (-0.4) = \mathbf{-0.2}$ [5]。

#### 步骤二：合成多个规则的结论

我们将分两步合成这三个值 $CF_1=0.48, CF_2=0.2, CF_3=-0.2$ [6]：

1.  **合成 $CF_1$ 与 $CF_2$（均为正值，适用“同正”公式）**：
    *   计算过程：$CF_{12} = 0.48 + 0.2 \times (1 - 0.48)$
    *   $CF_{12} = 0.48 + 0.2 \times 0.52$
    *   $CF_{12} = 0.48 + 0.104 = \mathbf{0.584}$
2.  **合成 $CF_{12}$ 与 $CF_3$（一正一负，适用“异号”公式）**：
    *   此时 $CF_{12} = 0.584$，$CF_3 = -0.2$。
    *   先计算分母项：$1 - \min(|0.584|, |-0.2|) = 1 - \min(0.584, 0.2) = 1 - 0.2 = 0.8$。
    *   计算分子项：$0.584 + (-0.2) = 0.384$。
    *   最终合成结果：$CF_{COMB} = \frac{0.384}{0.8} = \mathbf{0.48}$。

---

**最终计算结果：**
综合所有证据后，假设 $H$ 的最终确定性因子为 **0.48**。
