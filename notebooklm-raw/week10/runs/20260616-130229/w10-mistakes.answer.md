根据 Week 10 的课程笔记及相关参考书目，以下是深度学习优化与正则化中最易混淆的 5 组核心概念对比：

### 1. 优化器：Adam vs. SGD
这两者是神经网络训练中最常用的优化算法，代表了自适应学习率与固定学习率两种不同范式。

| 特性 | 随机梯度下降 (SGD) | Adam 优化器 |
| :--- | :--- | :--- |
| **核心原理** | 每次使用一小批样本的梯度更新参数 [1, 2]。 | 结合了动量法（一阶矩）和 RMSprop（二阶矩）的自适应学习率算法 [3, 4]。 |
| **学习率调节** | 所有参数共享同一个学习率，通常需要手动设置衰减策略 [1, 5]。 | 为每个参数动态调整学习率，能自动处理梯度稀疏或尺度差异问题 [3, 6]。 |
| **收敛特性** | 路径可能存在震荡，但最终泛化性能往往较好 [7, 8]。 | 初期收敛速度极快，是目前工业界最常用的“开箱即用”优化器 [3, 9]。 |
| **计算开销** | 计算简单，内存占用极低 [2]。 | 需要额外存储梯度的均值和方差，内存占用约为 SGD 的 3 倍 [4, 9]。 |

---

### 2. 数据预处理：白化 vs. 标准化
这两者都旨在改善损失函数景观（Landscape）的几何性质，降低优化难度。

| 特性 | 标准化 (Standardization) | 白化 (Whitening) |
| :--- | :--- | :--- |
| **处理方式** | 对每个维度独立进行零均值、单位方差处理 [10, 11]。 | 通过特征值分解（如 PCA 白化）将数据变换为各向同性的高斯分布 [10, 12]。 |
| **维度相关性** | **忽略**维度间的相关性，仅调整尺度 [10]。 | **消除**维度间的相关性，使协方差矩阵变为单位矩阵 [10, 13]。 |
| **几何影响** | 将损失曲面的轴对齐椭圆重塑为更接近圆形的形态 [10, 14]。 | 将损失曲面的狭长峡谷重塑为平滑圆碗，效果理论上最优 [10]。 |
| **计算复杂度** | 计算高效，适用于大规模高维数据 [10]。 | 计算极其昂贵（涉及协方差矩阵求逆），实际中较少直接使用 [10, 12]。 |

---

### 3. 参数初始化：Xavier vs. He 初始化
不当的初始化会导致梯度消失或爆炸，这两者是针对不同激活函数的最优解。

| 特性 | Xavier 初始化 (Glorot) | He 初始化 (Kaiming) |
| :--- | :--- | :--- |
| **适用对象** | 饱和激活函数：Sigmoid、Tanh [15, 16]。 | 非饱和激活函数：ReLU 及其变体 [15]。 |
| **数学推导基础** | 假设网络是线性的，旨在保持输入输出的方差一致 [15, 16]。 | 考虑到 ReLU 会将一半输入置零（方差减半），故在 Xavier 基础上翻倍 [15, 17]。 |
| **权重方差公式** | $Var(W) = \frac{2}{n_{in} + n_{out}}$ [15, 16]。 | $Var(W) = \frac{2}{n_{in}}$ [15]。 |
| **核心目标** | 防止深层网络前向传播时信号过早进入饱和区 [15, 18]。 | 补偿 ReLU 导致的方差损失，确保信号能传导至深层 [15, 17]。 |

---

### 4. 归一化：批量归一化 (BN) vs. 层归一化 (LN)
归一化是深层网络训练的“催化剂”，两者在执行的维度上存在本质区别。

| 特性 | 批量归一化 (BN) | 层归一化 (LN) |
| :--- | :--- | :--- |
| **归一化维度** | 在一个小批量（Batch）内，对**每一个特征**分别进行归一化 [10, 19]。 | 在**单个样本**内，对该层的所有神经元（Feature）进行归一化 [20]。 |
| **Batch 大小依赖** | 强依赖 Batch Size；若 Batch 太小，均值和方差估计会极不稳定 [1, 21]。 | **不依赖** Batch 大小，对单个样本即可执行 [20]。 |
| **适用场景** | 图像任务（CNN）表现卓越，通常置于卷积层之后 [10, 22]。 | 序列任务（RNN、Transformer）表现更好，处理变长序列更自然 [20, 23]。 |
| **主要解决问题** | 内部协变量偏移（Internal Covariate Shift）[10, 24]。 | 深层网络（特别是循环结构）中的梯度流稳定性 [20]。 |

---

### 5. 正则化：L2 正则化 vs. Dropout
这是防止模型过拟合（Overfitting）的两大核心手段。

| 特性 | L2 正则化 (权重衰减) | Dropout (随机失活) |
| :--- | :--- | :--- |
| **作用机制** | 在损失函数中添加参数平方项，惩罚过大的权重 [15, 25]。 | 训练时以概率 $p$ 随机将神经元的输出置为 0 [26, 27]。 |
| **直观理解** | 约束模型的复杂度，使预测函数更平滑 [15, 28]。 | 类似于集成学习，防止神经元之间产生过度的共适应 [26, 29]。 |
| **参数影响** | 使得权重倾向于变小，但不会变为 0（除非学习率极大）[25, 30]。 | 强制网络学习更具鲁棒性的特征，即使部分信号丢失也能正确识别 [27, 31]。 |
| **使用位置** | 通常只作用于权重矩阵 $W$，**偏置项 $b$ 通常不参与** [15, 32]。 | 作用于隐藏层神经元的激活值 [26, 27]。 |
