根据课件 07 (Logics and Prolog) 与 Week 14 的课堂记录，**消解反驳证明（Resolution Refutation Proof）** 是一种通过证明“知识库与目标的否定之集合”不可满足，从而验证原目标为真的自动推理技术 [1, 2]。

以下通过经典的 **“Fido 是否会死”** 例题演示完整流程：

### 1. 给定前提集合与待证目标
根据课件提供的逻辑断言 [1, 3]：
*   **前提 (1)**：所有的狗都是动物。 $\forall x (dog(x) \rightarrow animal(x))$
*   **前提 (2)**：所有的动物都会死。 $\forall y (animal(y) \rightarrow die(y))$
*   **前提 (3)**：Fido 是一只狗。 $dog(fido)$
*   **待证结论 (G)**：Fido 会死。 $die(fido)$

---

### 2. 化为子句集 (Clause Form)
根据 Week 14 记录的转化步骤（消去蕴含、略去量词、结论取反）[2]：

*   **子句 C1** (由前提 1 转化)：$\neg dog(x) \lor animal(x)$ [3]
*   **子句 C2** (由前提 2 转化)：$\neg animal(y) \lor die(y)$ [3]
*   **子句 C3** (由前提 3 转化)：$dog(fido)$ [3]
*   **子句 C4** (**将目标结论取反**)：$\neg die(fido)$ [2, 3]

此时，我们的目标是证明集合 $\{C1, C2, C3, C4\}$ 是 **不可满足的** [2]。

---

### 3. 合一 (Unification) 与消解 (Resolution) 过程
通过寻找互补文字（一正一负且可匹配），并利用变量替换（合一）进行推导 [2, 4]：

**第一步消解：**
*   选择 **C2** ($\neg animal(y) \lor die(y)$) 与 **C4** ($\neg die(fido)$)。
*   **互补文字**：$die(y)$ 与 $\neg die(fido)$。
*   **合一替换**：使用替换 $\theta = \{fido/y\}$ 使文字对齐 [4]。
*   **得到新子句 (C5)**：$\neg animal(fido)$。

**第二步消解：**
*   选择 **C1** ($\neg dog(x) \lor animal(x)$) 与 **C5** ($\neg animal(fido)$)。
*   **互补文字**：$animal(x)$ 与 $\neg animal(fido)$。
*   **合一替换**：使用替换 $\theta = \{fido/x\}$ [4]。
*   **得到新子句 (C6)**：$\neg dog(fido)$。

**第三步消解：**
*   选择 **C3** ($dog(fido)$) 与 **C6** ($\neg dog(fido)$)。
*   **互补文字**：$dog(fido)$ 与 $\neg dog(fido)$。
*   **消解结果**：产生 **空子句 $\square$** [4]。

---

### 4. 证明结论
由于消解过程最终产生了 **空子句 $\square$**，这表明子句集 $\{C1, C2, C3, C4\}$ 存在逻辑矛盾，是 **不可满足的** [2, 4]。
根据反驳原理，既然“结论的否定”导致了矛盾，则原命题 **“Fido 会死 (die(fido))” 必然为真** [2]。

---

### 补充：消解树图（文字版）
该树状图展示了从目标取反开始，不断与知识库子句“碰撞”的过程 [4]：

```text
       [ C2: ¬animal(y) ∨ die(y) ]      [ C4: ¬die(fido) ]
                    \                      /
                     \      {fido/y}      /
                      \__________________/
                               |
                    [ C5: ¬animal(fido) ]       [ C1: ¬dog(x) ∨ animal(x) ]
                               \                      /
                                \      {fido/x}      /
                                 \__________________/
                                          |
                                 [ C6: ¬dog(fido) ]       [ C3: dog(fido) ]
                                          \                      /
                                           \       { }          /
                                            \__________________/
                                                     |
                                               [ □ (空子句) ]
```
